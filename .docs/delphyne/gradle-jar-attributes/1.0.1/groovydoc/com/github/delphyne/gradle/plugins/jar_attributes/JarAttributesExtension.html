Package: com.github.delphyne.gradle.plugins.jar_attributes

[Groovy] Class JarAttributesExtension

    • com.github.delphyne.gradle.plugins.jar_attributes.JarAttributesExtension

    • Configuration for the JarAttributesPlugin. Configuration is done via builder syntax: 

      
 jarAttributes {
   applyDefaults false
   foo "bar"
   baz true
   someCalculatedValue = { Math.random() };
 }
      Will create a manifest with the following entries: foo: bar baz: true someCalculatedValue: -0.1234442112
